# GitHub Pages Only Version

Upload these files to GitHub:

- `index.html`
- `styles.css`
- `app.js`
- `README.md`

Then turn on GitHub Pages:

1. Open the repository in GitHub.
2. Go to `Settings`.
3. Go to `Pages`.
4. Source: `Deploy from a branch`.
5. Branch: `main`.
6. Folder: `/root`.
7. Save.

Your site will open at:

`https://YOUR-GITHUB-NAME.github.io/YOUR-REPO-NAME/`

Important: this version saves data in the browser on that device only. It is not a shared database.
